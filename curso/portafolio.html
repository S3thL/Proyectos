Habilidades especificas:


CSS3

Es el lenguaje que define el estilo y el diseño de una página web. CSS significa Cascading Style Sheets, que se traduce como Hojas de Estilo en Cascada. Un documento CSS está formado por reglas que especifican cómo se aplican los colores, las fuentes, los márgenes, el posicionamiento, las animaciones y otros efectos visuales a los elementos HTML.

HTML5

Es el lenguaje que define la estructura y el contenido de una página web. HTML significa HyperText Markup Language, que se traduce como Lenguaje de Marcado de Hipertexto. Un documento HTML está formado por etiquetas que indican cómo se organiza y muestra la información en el navegador.

Javascript

Es el lenguaje que define el comportamiento y la interactividad de una página web. JavaScript es un lenguaje de programación que permite crear y manipular dinámicamente los elementos HTML y CSS. Un documento JavaScript está formado por instrucciones que se ejecutan en el navegador cuando el usuario interactúa con la página web.
